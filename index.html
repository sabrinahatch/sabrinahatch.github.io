FCFS, SRPT, & PS Discrete Event Simulators

For my research project for Carnegie Mellon University's REUSE program, I studied the efficiency of routing protocols on data center networks. Specifically, I studied the effects of the first come first serve, shortest remaining processing time, and processor sharing scheduling policies for both the single server and single switch abstractions. To emulate their efficacy, I built DES programs in Python to model each policy's average job completion time based on varying factors such as system load, job size distribution, and congestion control implementation.

Simple Webserver in Python

As part of my final project for my computer networks class, I created a rudimentary webserver that could accept multiple HTTP requests at a time using multithreading. To test its constraints, I created my own HTTP client and deployed a DDoS attack against to examine the vunerabilites of unsecured servers.
